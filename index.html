Carrot is an esoteric "programming language" made for code-golf created by PPCG user Kritixi Lithos.

Credits: Carrot was inspired by TeaScript, CJam and Pyth. The code for primality checking is taken from Vihan's TeaScript source code. The Roboto Mono font was taken from Google Fonts.

Cheatsheet:

This is how every program in Carrot looks like:

stack^commands

The stack starts off as a String that can further be manipulated with the commands. The stack doesn't need to be enclosed in quotes. Another thing to note is the the stack is basically a String and not an array. Next up is the caret "^". This is required in every program as it seperates the stack from the commands. The commands manipulate the stack, to put it simply. At the termination of the program, the stack, after all the commands are applied, is printed to the output.

Learn Via Examples

Hello, World!^
This is a simple "Hello, World!" program. "Hello, World!" is pushed into the stack and is printed to the console. A shorter version would be:
Hello, World!

as^+"df"
This code prints "asdf" to the output. Here is how it works: first "as" is pushed onto the stack. Then it is concatenated (using the "+") with "df". Note how the stack doesn't require quotes but the commands do require them.

asdfghjkl^-3
This code prints "asdfgh" to the output. Here is how it works: first "asdfghjkl" is pushed onto the stack. Then the "-" operator performs a substring to the string. Because the number after the "-" is 3, the length of the stack is reduced by 3 as such: "asdfghjkl" - "jkl" = "asdfgh".

as^*1
The code prints "asas" to the output. Here is how it works: first "as" is pushed onto the stack. Then comes the "*" operator. It multiples the string by n+1 times where "n" is the number after the "*". Therefore, the stack "as" is multiplied by 1+1=2 times like so: "as"+"as"="asas".

0123^/2
The code prints "013" to the console. The "/" removes character at index 2(the number after the "/" sign).

List of Commands:

  • "+": contenation amongst strings and addition amongst integers
  • "-": substring|subtraction
  • "*": Multiplies stack by the succeeding number of times
  • "/": Removes char at given index|division
  • "i": Converts stack into an integer
  • "#" and "$": The string/int values of the input respectively
  • "P": primality checking
  • "?:|": if-else statements
  • "[]": repeat loop
